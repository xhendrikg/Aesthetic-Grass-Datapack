Aesthetic Grass
by xhendrikg
-----------------------------------------------------------------------------------
Adds:

    Custom Trigger that Places Grass on any Blocks
    Message when you did /reload

-----------------------------------------------------------------------------------
Usage:

    /datapack enable "file/Aesthetic Grass"
    /trigger aestheticseed


-----------------------------------------------------------------------------------
Tips:

    Use it to decorate your Bases Roof or
    Stone Blocks or
    A old house
